### Trace SDK — current state

`packages/sdk` (`@pulsechain-dev/trace-sdk`) is a standalone EVM trace toolkit.
Public-facing API surface:

**Types** (`src/types.ts`)
- Wire-format: `RawCallFrame`, `RawStructLog`, `RawCallType`
- Canonical: `TraceFrame` (bigint values, `children`, branded Address/Hex,
  `to: Address | null` for CREATE), `OpcodeStep`, `StateDiff`, `StorageChange`,
  `GasProfile`, `GasProfileEntry`, `DecodedParam`, `TraceResult`, `CallType`
- Consumer aliases: `CallNode = TraceFrame`

**Loaders** (`src/loaders/`)
- `loadTraceFromObject({ callFrame, structLogs?, txHash?, blockNumber? })`
- `loadTraceFromFile(jsonString)`
- `loadTraceFromHash({ txHash, rpcUrl, fetch?, timeoutMs? })`
- `normalizeCallFrame` / consumer alias `parseCallTrace`
- `normalizeStructLogs`

**Traversal** (`src/traversal/`) — all explicit-stack, safe on 50k-deep trees
- `walkCallTree(frame, { enter?, exit? })` — `enter` can return `false` or `"skip"`
- `flattenCallTree(frame)` → `FlatFrame[]` with pre-order indices
- `filterByAddress(frame, address, options)`
- `filterBySelector(frame, selector)`
- `findRevertFrame(frame)` — innermost reverter
- `buildGasProfile(frame)` — bottom-up `selfGas` reduction

**Components** (`src/components/`)
- `<CallTree frame={…} onSelect={…} classNames={…} />`
- `<GasFlamegraph frame={…} resolveSelector={…} classNames={…} />`
- `<OpcodeViewer steps={…} onSelectStep={…} classNames={…} />`
- Pure helpers also exported: `truncateAddress`, `formatGas`, `formatWei`,
  `getFunctionSelector`, `classifyOpcode`, `getOpcodeColor`, `isExpensiveOp`,
  `OPCODE_CATEGORY_COLORS`, `buildFlamegraphLayout`, `adjustBrightness`,
  `getBarColor`
- All components use sensible default colors + per-slot `classNames` for theming

**Util** (`src/util/`)
- `describeUnknownError(err)` — extracted catch-block helper

**Test infrastructure**
- vitest + jsdom + @testing-library/react + @vitest/coverage-v8
- 157 tests pass; coverage gated at **100/100/100/100** via vitest thresholds
- Coverage threshold enforcement IS active — drops fail `npm run test --workspace=packages/sdk`
- Test fixtures: `tests/fixtures.ts` (`sampleRawCallFrame`, `makeFrame`, `addrs`)

**Wiring into web**
- `packages/web` depends on `@pulsechain-dev/trace-sdk: "*"` (npm workspace link)
- `DebuggerView.tsx` imports `CallTree`, `GasFlamegraph`, `OpcodeViewer`,
  `normalizeCallFrame` from SDK; normalizes wire-format CallFrame at the
  render boundary via `useMemo(() => normalizeCallFrame(callTrace), [callTrace])`
- For OpcodeViewer the Hex-vs-string nominal mismatch is bridged with
  `as unknown as Parameters<typeof OpcodeViewer>[0]["steps"]` — runtime
  identical, type system reconciliation deferred until the web API client
  is migrated to import types from SDK

### Outstanding work — explicit consumer queue

Per user requests captured in memory
(`sdk_consumer_api_contract.md`, `sdk_findings_features.md`):

**1. Remaining component extractions**
   - `StepDebugger` — step-by-step opcode navigator (couples to OpcodeViewer
     via shared step index; likely extract small `useOpcodeNavigation` hook)
   - `SourceViewer` — most complex; depends on source-map + recompile data
     from the API. Should ship as data-agnostic with `source: string` +
     `highlightLine: number` props
   - `StateDiffPanel`, `FrameDetailPanel` — not explicitly requested but
     listed in original SDK design notes

**2. Pure-parser additions (consumer API contract)**
   - `parseTokenDeltas(callTree): TokenDelta[]` — extract ERC-20 Transfer
     events. Requires extending `RawCallFrame` with optional
     `logs?: RawLog[]` (from callTracer `withLog: true` mode) OR accepting
     logs as a sibling input
   - `parsePrestateDiff(raw): BalanceDelta[]` — parse prestateTracer
     diffMode JSON. Independent of call tree
   - Both pure functions, no I/O, easy to test at 100%

**3. Risk analyzer + UI**
   - `analyzeRisks(callTree, whitelist: Set<Address>): RiskFlag[]`
   - Initial rule types: `DELEGATECALL_UNRECOGNIZED`,
     `TOKEN_SENT_TO_TOKEN_CONTRACT`, eventually `LARGE_APPROVAL`
   - `<FindingsPanel risks={…} />` component to render the output
   - Need a heuristic or registry for "is this address a token contract?"
     (heuristic: `eth_getCode` non-empty + `balanceOf(address)` selector)

**4. Web-side cleanup (small)**
   - Migrate `packages/web/src/api/debugger.ts` `OpcodeStep` to import from
     SDK, eliminating the `as unknown as` cast in DebuggerView
   - Same for `CallFrame` — would let us drop the `normalizeCallFrame` step
     for already-canonical data (though API server returns wire format so
     normalization at boundary still needed; this is more about type
     compatibility than runtime)

### Blockers / gotchas (unchanged)

- Public PulseChain RPC slow for complex traces. Use the private RPC in `.env`.
- Source map recompilation downloads solc on first use (~3s), then caches.
- Anvil must be installed (Foundry toolchain) for fork-based simulation.
