#### Public-facing API surface

**Types** (`src/types.ts`)
- Wire-format: `RawCallFrame` (with `logs?: RawLog[]`), `RawLog`,
  `RawStructLog`, `RawCallType`, `RawPrestateAccount`, `RawPrestateDiff`
- Canonical: `TraceFrame` (with `logs?: Log[]`), `Log`, `OpcodeStep`,
  `StateDiff`, `StorageChange`, `GasProfile`, `GasProfileEntry`,
  `DecodedParam`, `TraceResult`, `CallType`
- Parser outputs: `TokenDelta`, `BalanceDelta`
- Risk types: `RiskFlag`, `RiskSeverity`, `RiskFlagType`,
  `AnalyzeRisksOptions`
- Consumer aliases: `CallNode = TraceFrame`

**Loaders** (`src/loaders/`) — unchanged
- `loadTraceFromObject`, `loadTraceFromFile`, `loadTraceFromHash`
- `normalizeCallFrame` / alias `parseCallTrace`, `normalizeStructLogs`
- `normalizeCallFrame` now passes `logs` through (lowercased addresses,
  preserved topics/data)

**Traversal** (`src/traversal/`) — unchanged
- `walkCallTree`, `flattenCallTree`, `filterByAddress`, `filterBySelector`,
  `findRevertFrame`, `buildGasProfile`

**Parsers** (`src/parsers/`) — NEW in 0.1.1
- `parseTokenDeltas(frame) → TokenDelta[]` — ERC-20 Transfer extractor.
  Walks tree, skips reverted subtrees (their logs are rolled back).
  Disambiguates ERC-20 (3 topics) from ERC-721 (4 topics) by topic count.
- `parsePrestateDiff(raw) → BalanceDelta[]` — net ETH balance changes
  from prestateTracer diffMode payload. Three-way semantic for missing
  post entries (absent address = self-destruct, undefined .balance =
  unchanged, defined = new value). Sorted by address ascending.

**Risks** (`src/risks/`) — NEW (committed, in 0.1.1+, not yet published)
- `analyzeRisks(frame, options?) → RiskFlag[]` — pure call-tree walker.
  Walks INTO reverted subtrees and tags findings with `reverted: true`
  rather than skipping (security audits want "what almost happened").
  Maintains a reverted-ancestor stack via enter/exit hooks.
- Built-in rule: `delegatecallUnrecognized` — flags any DELEGATECALL
  whose target is not in the optional `whitelist` Set.
- `BUILTIN_RULES` and `RiskRule` type exported for custom rule composition.
- Rules return `Omit<RiskFlag, "reverted">` — analyzer owns that field.

**Components** (`src/components/`)
- `<CallTree>` — interactive call tree (existing)
- `<GasFlamegraph>` — proportional gas-cost flamegraph (existing)
- `<OpcodeViewer>` — step-by-step opcode trace (existing)
- `<FindingsPanel>` — renders `RiskFlag[]` grouped by severity (NEW)
- `<StateDiffPanel>` — renders `StateDiff[]` with balance/nonce/code/
  storage changes per address (NEW)
- `<FrameDetailPanel>` — single-frame metadata view with decoded I/O,
  error banner, function selector (NEW)
- Pure helpers also exported: `truncateAddress`, `formatGas`, `formatWei`,
  `getFunctionSelector`, `classifyOpcode`, `getOpcodeColor`, `isExpensiveOp`,
  `OPCODE_CATEGORY_COLORS`, `buildFlamegraphLayout`, `adjustBrightness`,
  `getBarColor`

**Util** (`src/util/`)
- `describeUnknownError(err)` — extracted catch-block helper

**Test infrastructure**

- 252 tests pass; coverage gated at **100/100/100/100** via vitest thresholds

**Wiring into web**

### Blockers / gotchas (unchanged)

- Public PulseChain RPC slow for complex traces. Use the private RPC in `.env`.
- Source map recompilation downloads solc on first use (~3s), then caches.
- Anvil must be installed (Foundry toolchain) for fork-based simulation.
